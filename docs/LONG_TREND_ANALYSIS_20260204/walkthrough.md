# 長期トレンド予測 (Long Trend Predictor) 実装完了

池水雄一氏の分析手法に基づき、長期的な視点での貴金属分析およびポートフォリオ推奨を行うモジュールを実装しました。

## 実装内容

### 新規モジュール: `metal_analyzer/models/long_trend_predictor.py`
このモジュールは、以下の4つのダッシュボード形式で月足データおよびマクロ経済指標を分析します。

| ダッシュボード | コンセプト (池水メソッド) | 実装ロジック |
| :--- | :--- | :--- |
| **1. 通貨価値** | **通貨価値の毀損** | ドル建て金価格の長期上昇トレンドと、ドルインデックス (DXY) の下落傾向を確認します。 |
| **2. 相対価値** | **割安資産の発掘** | 金銀レシオ (GSR) や金プラチナレシオを計算し、歴史的に割安な貴金属を特定します。 |
| **3. マクロ環境** | **実質金利** | TIPS (米国物価連動国債) のトレンドを使用し、実質金利が金に有利な状態（低下傾向）にあるかを判定します。 |
| **4. ポートフォリオ** | **資産保全** | 上記の分析結果に基づき、推奨される貴金属の保有比率（5% 〜 25%）を提示します。 |

### 検証スクリプト: `examples/verify_long_trend.py`
実際の市場データ（Yahoo Financeから取得）を使用して、新機能の動作を確認するスクリプトです。取得データ：金(GC=F), 銀(SI=F), プラチナ(PL=F), ドルインデックス(DX-Y.NYB), TIPS(TIP).

## 検証結果 (2026/02/04実施)

```text
--- 分析結果 ---
ダッシュボード1 (通貨価値): 長期上昇 (通貨価値下落) / ドル安トレンド (金に追い風)
ダッシュボード2 (相対価値): 銀の割安感解消 (GSR: 58.3), プラチナ超割安 (倍率: 2.2)
ダッシュボード3 (マクロ環境): 実質金利低下傾向 (金に強力な追い風)
ダッシュボード4 (ポートフォリオ): 20-25% (積極投資)

総合コメント:
全指標が好転しています。法定通貨のリスクヘッジとして、そして値上がり益を狙う資産として最大級の組み入れを推奨します。銀やプラチナへの分散も効果的です。
```

- **マクロ環境**: ドル安・実質金利低下という、金にとって理想的な環境を正しく検知しました。
- **割安判定**: プラチナが金に対して「超割安」であると判定されました。
- **ポートフォリオ**: これを受けて、最大比率である「20-25%」の投資推奨が出力されました。
